# RHYTHM SIMULATOR
## Description
The Rhythm Simulator is a program that simulates a rhythm game’s patterns 
to practice and understand the fundamentals of rhythm games. Users can use 
the app together with another person and essentially play the game as a medium 
to practice with.

## How To Run
1. Have Python Installed (Python 3.13)
2. Have Pygame imported using 'pip install pygame'
3. Program Structure
```
RhythmSim/
├── main.py
├── level1.py
├── assets
│   ├── blue_right_arrow.png
│   ├── gblue_right_arrow.png
│   ├── ggreen_down_arrow.png
│   ├── gred_left_arrow.png
│   ├── green_down_arrow.png
│   ├── gyellow_up_arrow.png
│   ├── red_left_arrow.png
│   └── yellow_up_arrow.png
```
4. Run the program (main.py)
